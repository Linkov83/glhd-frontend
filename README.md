# Global Laser Hardening Database (GLHD)

A web-based interface for exploring, analyzing, and contributing to a database of laser hardening experiments.

## 🌐 Live Demo
[https://glhd-frontend.vercel.app](https://glhd-frontend.vercel.app)

## 🛠 Tech Stack
- React + TailwindCSS
- Django + DRF (backend)
- PostgreSQL (DB)
